RLTK's documentation
=====================

Record Linkage ToolKit.

Getting Start
--------------

Installation (need to upload to PYPI later)::

   pip install rltk

Example::

   >>> import rltk
   >>> rltk.levenshtein_distance('abc', 'abd')
   1
